# Go Day 01 - Student Profile

My first Go mini project.

## Description

This project displays a student's profile in the terminal using basic Go concepts.

## What I Practiced

- Variables
- Constants
- Functions
- Multiple Return Values
- Named Return Values
- Type Conversion
- fmt.Println()

## Technologies

- Go 1.26

## Project Structure

```
go-day01-student-profile/

├── main.go
└── README.md
```
